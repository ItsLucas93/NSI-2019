L'histoire des langages de programmations moderne.

Voici l'histoire des langages de programmations les plus utilisés à notre époque.

• C – 1972, Co-développé par Dennis Ritchie, Bell Labs

C'est un langage de programmation généraliste, de bas niveau (C'est à dire qui ne fournit que peu d'abstraction par rapport au à la capacité du processeur de la machine d'exécuter une tâche). Inventé au début des années 1970, C’est encore l'un des langages les plus utilisés de nos jours.

• Python – 20 Février 1991, par le néerlandais Guido van Rossum

Python est un langage de programmation interprété (ayant pour tâche d'analyser, de traduire et d'exécuter les programmes écrits dans un langage informatique.) et multiplateformes. Il favorise la programmation impérative structurée, fonctionnelle et orientée objet. Le langage Python est placé sous une licence libre et fonctionne sur la plupart des plates-formes informatiques, des smartphones aux ordinateurs centraux11, de Windows à Unix avec notamment GNU/Linux en passant par macOS. Il est conçu pour optimiser la productivité des programmeurs en offrant des outils de haut niveau et une syntaxe simple à utiliser.

• HTML – 1993, Organisation World Wide Web Consortium (W3C)

L’HyperText Markup Language, généralement abrégé HTML, est le langage de balisage conçu pour représenter les pages web. HTML permet également de structurer et de mettre en forme le contenu des pages, d’inclure des ressources multimédias dont des images, des formulaires de saisie et des programmes informatiques. Il permet de créer des documents interopérables avec des équipements très variés de manière conforme aux exigences de l’accessibilité du web. Il est souvent utilisé conjointement avec le langage de programmation JavaScript et CSS.

• PHP – 1994, Rasmus Lerdorf

PHP est un langage de programmation principalement utilisé pour produire des pages Web dynamiques via un serveur HTTP, mais pouvant également fonctionner comme n'importe quel langage interprété de façon locale. PHP est un langage impératif orienté objet. PHP a permis de créer un grand nombre de sites web célèbres, comme Facebook, Wikipédia, Il est considéré comme une des bases de la création de sites web dits dynamiques mais également des applications web.

• Java – 23 mai 1995, James Gosling et Patrick Naughton

Java est un langage de programmation orienté objet créé par James Gosling et Patrick Naughton, présenté officiellement le 23 mai 1995 au SunWorld. La société SunWorld a été ensuite rachetée en 2009 par la société Oracle qui détient et maintient désormais Java. La particularité et l'objectif central de Java est que les logiciels écrits dans ce langage doivent être très facilement portables sur plusieurs systèmes d’exploitation tels que Unix, Windows, Mac OS ou GNU/Linux, avec peu ou pas de modifications, mais qui ont l'inconvénient d'être plus lourd à l'exécution (en mémoire et en temps processeur) à cause de sa machine virtuelle

• JS – 4 Décembre 1995, Brendan Eich

JavaScript est un langage de programmation de scripts principalement employé dans les pages web interactives mais aussi pour les serveurs avec l'utilisation de Node.js. JavaScript a été créé en 1995 par Brendan Eich. Avec les technologies HTML et CSS, JavaScript est parfois considéré comme l'une des technologies cœur du World Wide Web5. Le langage JavaScript permet des pages web interactives, et à ce titre est une partie essentielle des applications web.

• CSS – 17 Décembre 1996, World Wide Web Consortium (W3C)

Les feuilles de style en cascade, généralement appelées CSS, forment un langage informatique qui décrit la présentation des documents HTML et XML. Les standards définissant CSS sont publiés par le World Wide Web Consortium (W3C). Introduit au milieu des années 1990, CSS devient couramment utilisé dans la conception de sites web et bien pris en charge par les navigateurs web dans les années 2000.

• Swift - 2014, Apple

Le Langage Swift est un langage de programmation développé par Apple dont le début de développement a été commencé en 2010, puis publié lors de la WWDC (Conférence des développeurs du monde entier organisé par Apple) en 2014, ayant pour objectif d’être :

  • Expressif (Swift offre une syntaxe agréable à utiliser, avec les fonctionnalités modernes attendues par les développeurs.),
  • Rapide (Swift est conçu pour remplacer les langages basés sur C (C, C ++ et Objective-C))
  • Sûr (Un comportement indéfini est l'ennemi de la sécurité et les erreurs de développement doivent être détectées avant que le logiciel ne soit en production).

    • Swift est officiellement supporté sur les systèmes d'exploitation Linux Ubuntu, iOS, iPadOS, macOS, watchOS et tvOS.

    (Version actuelle : 5.0 | 5.1 en cours de dev…)

    C’est en 2015 que Swift devient alors en Open-Source, c’est-à-dire libre d’accès au code source par tout le monde (swift.org). Grâce à cela, la communauté contribue à son amélioration pour que Swift soit un langage simple, efficace, pratique et durable dans le temps.
    L’apprentissage est accessible à tous, il existe différentes manières proposées par Apple pour pratiquer le langage Swift :

  • Playground | Application d’apprentissage interactif pour apprendre le Swift proposé par Apple
  • XCode | l’IDE proposé par Apple

    • La syntaxe et les conventions d’écritures rendent le code plus compacté, simplifié et donc plus rapide. D’après Apple, Swift serait 3x plus rapide que le langage python !
